Java Bhaskara
====================

Example Application: bhaskara

## Calculate Delta
> delta = b^2 -4 (a)(c)	

```java
delta = (Math.pow(b, 2)) - 4 * (a) * (c);
```

## Calculate X

#### Calculate x'
> x' = ( -b + sqrt(delta) )/(2 * a)

```java
x1 = (((b) * -1) + Math.sqrt(delta) / (2 * a);
```

#### Calculate x''
> x'' = ( -b - squareRoot(delta) )/(2 * a)

```java
x2 = (((b) * -1) - Math.sqrt(delta) / (2 * a);
```
